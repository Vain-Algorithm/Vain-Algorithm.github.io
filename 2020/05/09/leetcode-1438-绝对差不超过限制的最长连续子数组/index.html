Leetcode 1438. 绝对差不超过限制的最长连续子数组

Posted on 2020-05-09
leetcode

传送门：

题意：

给长度为 n 的序列 和 限制值 k，求最长的子区间，子区间满足区间中任意两个数的差的绝对值不大于k

思路：
双单调队列，一个维护当前区间最大，一个维护当前区间最小，如果当前区间最大-最小满足条件，呢么当前区间整体满足条件，否则让单调队列中的除去当前存在的左端点即可
